📘 ПОДРОБНАЯ ИНСТРУКЦИЯ: ПОЛУЧЕНИЕ API-КЛЮЧА И ID КАТАЛОГА YANDEX CLOUD

  1. Перейдите в консоль Yandex Cloud и авторизуйтесь.
  2. Выберите или создайте каталог (folder). Его ID можно скопировать из URL (после /folders/) или из карточки каталога. Как узнать ID каталога.
  3. Создайте сервисный аккаунт с ролью ai.languageModels.user:
    • В меню слева выберите «Сервисные аккаунты» (прямая ссылка).
    • Нажмите «Создать сервисный аккаунт», укажите имя (например, vk-bot-sa).
    • После создания перейдите в него, нажмите «Добавить роль», найдите ai.languageModels.user и добавьте.
  4. Создайте API-ключ для этого сервисного аккаунта:
    • На странице сервисного аккаунта перейдите на вкладку «API-ключи» или сразу после создания будет предложено создать ключ.
    • Нажмите «Создать API-ключ», введите описание, скопируйте сгенерированный ключ (показывается один раз). Документация.
  5. Вставьте скопированный ключ в поле «🔑 API ключ Yandex Cloud» выше, а ID каталога — в поле «📁 ID каталога».

📱 НАСТРОЙКИ VK

📘 ПОДРОБНАЯ ИНСТРУКЦИЯ: НАСТРОЙКА CALLBACK API ВКОНТАКТЕ

  1. Перейдите в управление сообществом ВК:
    • Откройте своё сообщество → нажмите «Управление» (или перейдите по ссылке https://vk.com/clubXXXX?act=edit, заменив XXXX на ID сообщества).
  2. В левом меню выберите «Работа с API» → «Callback API». Прямая ссылка: https://vk.com/clubXXXX?act=api (замените XXXX на ID).
  3. Нажмите «Добавить сервер» и заполните поля:
    • Название сервера: например, «Мой бот».
    • Адрес сервера: https://ваш-домен/api/callback (если тестируете локально, используйте ngrok).
    • Версия API: 5.131 (или новее).
  4. После добавления сервера скопируйте Строку подтверждения (это токен). Вставьте её в поле «🔐 Подтверждающий токен» выше.
  5. Нажмите «Подтвердить» — ВК отправит запрос на ваш сервер. Если сервер настроен правильно, он ответит этой строкой, и статус станет «Подтверждён».
  6. В разделе «Типы событий» отметьте необходимые:
    • Обязательно: «Сообщение входящее» и «Подтверждение сервера».
    • Остальные — по желанию.
  7. Сохраните настройки.

🔗 Полезные ссылки:

  • Документация VK Callback API
  • Генератор токена доступа сообщества (vkhost)
